# 🚀 방학 계획서 입력 시스템 원클릭 배포 가이드

본 가이드는 제공된 백엔드(`Code.gs`)와 프론트엔드(`Index.html`) 코드를 활용하여 **단 2분 만에 나만의 구글 스프레드시트 연동 웹앱을 무료로 배포하고 실서비스하는 방법**을 안내합니다.

## 🏃‍♂️ 3단계 초고속 배포 프로세스

### 1단계: 구글 스프레드시트 및 Apps Script 프로젝트 생성
1. [구글 스프레드시트 홈](https://sheets.google.com)으로 이동하여 **새 스프레드시트(빈 시트)**를 생성합니다.
2. 스프레드시트 상단 메뉴에서 **[확장 프로그램] ➔ [Apps Script]**를 클릭합니다.
3. 기본 생성된 코드 창에 입력되어 있는 `function myFunction() {}` 내용을 모두 지웁니다.
4. 준비해 둔 **`Code.gs`**의 전체 소스코드를 복사하여 붙여넣습니다.

### 2단계: 프론트엔드 Index.html 파일 생성 및 코드 병합
1. Apps Script 편집기 좌측 메뉴의 **[파일] 옆 [+] 버튼**을 누르고 **[HTML]**을 선택합니다.
2. 새 파일의 이름 입력란에 **`Index`**를 입력하고 엔터를 누릅니다 (자동으로 `Index.html` 파일이 생성됩니다).
3. 기본 생성된 템플릿 코드들을 모두 지웁니다.
4. 준비해 둔 **`Index.html`**의 전체 소스코드를 복사하여 붙여넣습니다.
5. 상단 메뉴바의 **[저장(Ctrl + S / 디스크 아이콘)]** 버튼을 눌러 프로젝트를 저장합니다.

### 3단계: 웹 앱 배포 및 권한 승인 (가장 중요 ⭐)
1. 편집기 우측 상단의 주황색 **[배포] ➔ [새 배포]** 버튼을 클릭합니다.
2. 기어 모양 설정 아이콘을 누르고 유형으로 **[웹 앱]**을 선택합니다.
3. 배포 설정을 아래와 같이 세팅합니다:
   - **설명**: `방학 계획서 입력 시스템 v1.0`
   - **다음 사용자로 실행**: **`나(내 구글 계정)`** (교직원들이 각자 구글 로그인 없이 바로 제출할 수 있게 하려면 반드시 '나'로 해야 합니다.)
   - **액세스 권한이 있는 사용자**: **`모든 사용자`** (로그인하지 않은 외부인이나 동료 교사도 양식을 채울 수 있게 하려면 반드시 '모든 사용자'로 지정해야 합니다.)
4. 하단의 **[배포]** 버튼을 클릭합니다.
5. **[액세스 승인]** 파란색 버튼이 나타나면 클릭합니다.
6. 자신의 구글 계정을 선택한 후, 경고 창이 나타나면 좌측 하단의 **[Advanced(고급)] ➔ [Go to ... (unsafe) / ... (으)로 이동]** 링크를 클릭합니다.
7. 우측 하단의 **[Allow(허용)]** 버튼을 눌러 스프레드시트 제어 권한을 허가합니다.
8. 완성된 **[웹 앱 URL]** 주소를 복사하여 교직원 메신저나 공지사항에 공유하면 즉시 실시간 입력이 개시됩니다!
