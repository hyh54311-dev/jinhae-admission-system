# 🏆 진해고등학교 입학홍보단 단체 문자 발송 시스템 가이드

이 문서에는 구글 메시지 웹(Google Messages Web)과 플레이라이트(Playwright) 자동화를 이용해 합격생들에게 1:1 개인 맞춤 안내 문자를 안전하고 빠르게 대량 발송하는 모든 과정이 정리되어 있습니다.

다음에 다시 발송할 일이 있을 때, 이 가이드를 보고 실행하면 **오류 없이 한 번에 발송을 끝마치실 수 있습니다.**

---

## 🚀 3초 만에 발송 시작하기 (가장 간편한 방법)

폴더 내에 새롭게 생성된 **`진해고_입학홍보단_문자발송기.bat`** 파일을 더블 클릭하여 실행해 주세요. 
메뉴 인터페이스가 표시되며, 숫자 입력만으로 모든 단계가 순차 진행됩니다.

1. **`1` 입력 후 Enter**: 발송 전용 독립 크롬 창 실행
2. 크롬 창이 뜨면 **구글 메시지 웹**에 접속해 스마트폰으로 QR 코드를 한 번만 스캔하여 기기 페어링을 완료합니다.
3. **`2` 입력 후 Enter**: 자동 1:1 대량 문자 발송 시작!

---

## 🛠️ 시스템 상세 동작 원리

이 프로그램은 사용자의 스마트폰과 연동된 구글 메시지 웹을 크롬의 **디버깅 포트(CDP - 9222번 포트)**를 통해 제어합니다.

### 1. 왜 독립된 크롬 창(`Chrome_디버그모드.bat` / 메뉴 1번)을 사용하나요?
일반 크롬 창을 사용하면 기존에 열어둔 탭들이 디버그 모드 차단 정책에 걸려 디버깅 포트가 켜지지 않습니다. 
따라서 본 프로그램은 **작업 중인 크롬 창들을 전혀 닫지 않고도** 동시에 실행할 수 있는 독립된 디버그 전용 크롬 프로필(`%LOCALAPPDATA%\Google\Chrome\Antigravity_Debug_Session`)을 생성해 작동합니다.
* **장점**: 로그아웃되거나 다른 탭이 꺼질 걱정이 없고, **최초 1회만 기기 연동(QR 스캔)**을 해두면 세션이 안전하게 저장되어 다음 실행 시에는 페어링 없이 바로 발송이 가능합니다.

### 2. 개인정보 보호를 위한 1:1 개별 발송 방식
단체 MMS를 보낼 경우 수신자 전원의 전화번호가 서로에게 노출되어 **심각한 개인정보 침해 및 사생활 노출 우려**가 있습니다.
본 자동화 프로그램은 학생 한 명 한 명의 채팅방을 개별적으로 열어 **독립된 1:1 문자**를 자동으로 발송하므로, 학생들의 정보 노출이 원천 차단됩니다.

## ✏️ 나중에 문자 내용을 바꾸고 싶을 때 (메시지 템플릿 수정)

합격자 발표 외에 **소집 장소 변경, 추가 안내사항 전달 등** 문자 내용을 변경하고 싶다면 다음 방법을 이용하세요.

1. 폴더 내의 **`send_bulk_sms.py`** 파일을 마우스 우클릭하여 **[연결 프로그램] -> [메모장]**으로 엽니다.
2. 상단 14~31라인 근처에 정의된 `MESSAGE_TEMPLATE` 내용을 수정합니다.
   * **주의**: `{이름}`과 `{학번}` 부분은 실제 학생 정보로 자동 변환되는 자리표시자(Placeholder)입니다. 지우지 말고 그대로 활용해 주세요.
   * **예시**:
     ```python
     MESSAGE_TEMPLATE = """[진해고등학교 홍보단 긴급 안내]
     안녕하세요 {이름}({학번}) 학생, 홍보단 담당 교사입니다.
     내일 소집 장소가 가온실에서 '시청각실'로 변경되었으니 참고해 주세요."""
     ```
3. 저장 후 문자발송기에서 2번을 눌러 실행하시면 즉시 수정된 내용으로 발송됩니다.

## 🩺 오류 해결 (트러블 슈팅)

### ❌ `ECONNREFUSED 127.0.0.1:9222` 에러가 납니다.
* **원인**: 프로그램을 돌릴 때 디버그용 크롬 창(메뉴 1번)이 켜져 있지 않거나, 연결 포트가 차단되었을 때 발생합니다.
* **해결**: 문자발송기 메뉴에서 **`1`번을 선택해 크롬 창을 반드시 먼저 실행**해 두고 발송을 시작해 주세요.

### ❌ 스마트폰과 연결하라는 QR 코드가 계속 뜹니다.
* **원인**: 1번으로 실행한 독립 크롬 창에서 최초 기기 페어링(QR코드 스캔)이 완료되지 않았거나 연결이 풀린 상태입니다.
* **해결**: 크롬 창의 화면에 나온 QR 코드를 스마트폰의 구글 메시지 앱의 **"기기 페어링"** 기능을 켜서 한 번만 카메라로 찍어주시면 즉시 연결됩니다.

### ❌ 학생 번호가 바뀌었거나 잘못 기재되었습니다.
* **해결**: 구글 드라이브의 원본 스프레드시트에서 전화번호를 수정해 주시거나, 로컬 폴더 안의 `C:\Users\admin\.gemini\antigravity\brain\48fcecfb-78c6-4a7f-a6c5-40fca9cea47b\scratch\sheet_content.csv` 파일을 직접 메모장으로 열어 해당 학생의 번호를 수정한 후 다시 발송해 주시면 됩니다.
