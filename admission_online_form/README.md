# 2027학년도 진해고등학교 신입생 입학등록확인서 온라인 제출 시스템 구축 가이드

이 가이드는 학부모로부터 스마트폰 터치 서명을 포함한 **입학등록확인서**를 온라인으로 받아, 구글 문서 양식과 결합한 뒤 **PDF 파일로 자동 생성하여 구글 드라이브에 저장**하는 웹앱 시스템 구축 단계별 매뉴얼입니다.

---

## 1. 준비 작업 (구글 드라이브)

### ① 가정통신문 형식의 구글 문서(템플릿) 만들기
1. 구글 드라이브에서 새 **구글 문서(Google Docs)**를 생성하고, 학교의 공식 입학등록확인서 양식(가정통신문 형태)을 디자인합니다.
2. 학부모가 입력한 값과 제출 날짜가 자동으로 채워질 위치에 아래와 같이 **중괄호 두 개 `{{ }}`**로 묶인 템플릿 태그를 입력합니다.
   * 학생 성명: `{{학생이름}}`
   * 생년월일: `{{생년월일}}`
   * 학부모 성명: `{{학부모이름}}`
   * 학생과의 관계: `{{관계}}`
   * 연락처: `{{연락처}}`
   * 제출일 날짜: `{{년}}`년 `{{월}}`월 `{{일}}`일
   * 서명 표시 위치: `{{서명}}` (서명이 들어갈 수 있게 앞뒤로 적당한 여백을 둡니다.)
3. 생성한 구글 문서의 URL 주소에서 **문서 ID**를 복사하여 메모장 등에 기록해 둡니다.
   * 예: `https://docs.google.com/document/d/1A2B3C_document_id_XYZ/edit` 에서 **`1A2B3C_document_id_XYZ`** 부분이 문서 ID입니다.

### ② PDF 저장용 구글 드라이브 폴더 만들기
1. 구글 드라이브에 학부모들이 제출한 PDF 파일들이 모여 저장될 **새 폴더**를 만듭니다 (예: `2027학년도 신입생 입학등록확인서 제출`).
2. 해당 폴더로 들어가 URL 주소에서 **폴더 ID**를 복사하여 메모해 둡니다.
   * 예: `https://drive.google.com/drive/folders/1a2b3c_folder_id_xyz` 에서 **`1a2b3c_folder_id_xyz`** 부분이 폴더 ID입니다.

---

## 2. 구글 앱스 스크립트(GAS) 설정

### ③ 스프레드시트 및 스크립트 에디터 열기
1. 구글 드라이브에서 새 **구글 스프레드시트(Google Sheets)**를 하나 만듭니다 (예: `입학등록확인서 제출 명단`).
2. 상단 메뉴에서 **확장 프로그램** ➔ **Apps Script**를 클릭하여 개발 에디터 창을 엽니다.

### ④ 소스 코드 작성
1. 에디터 좌측의 `코드.gs` 파일 내용을 전부 지우고, 이 폴더의 **`Code.gs`** 소스 코드를 그대로 복사하여 붙여넣습니다.
2. 코드 상단의 설정 변수 부분을 수정합니다.
   ```javascript
   var TEMPLATE_DOC_ID = "여기에_아까_복사한_구글_문서_ID_붙여넣기"; 
   var SAVE_FOLDER_ID = "여기에_아까_복사한_폴더_ID_붙여넣기";
   ```
3. 에디터 좌측 상단의 **파일 추가(+)** 버튼을 클릭하고 **HTML**을 선택한 뒤, 파일 이름을 반드시 **`Index`**로 입력합니다. (확장자 `.html`은 자동으로 붙습니다.)
4. 새로 만든 `Index.html` 파일의 기본 내용을 모두 지우고, 이 폴더의 **`Index.html`** 소스 코드를 그대로 복사하여 붙여넣습니다.
5. 상단의 **프로젝트 저장(디스켓 모양 아이콘)** 버튼을 클릭하여 저장합니다.

---

## 3. 웹앱 배포 (학부모 배포용 링크 생성)

### ⑤ 웹앱 배포 설정
1. 에디터 우측 상단의 **배포** ➔ **새 배포** 버튼을 클릭합니다.
2. 새 배포 창 왼쪽 위의 톱니바퀴 아이콘(유형 선택)을 클릭하고 **웹앱(Web app)**을 선택합니다.
3. 배포 설정 옵션을 다음과 같이 선택합니다.
   * **설명:** `2027학년도 입학등록확인서 제출 웹앱`
   * **다음 사용자 권한으로 실행:** **`웹앱을 액세스하는 사용자`**가 아닌 **`나(본인 이메일 주소)`**로 설정합니다. (그래야 학부모들이 구글 로그인을 하지 않고도 작성할 수 있으며, 내 권한으로 구글 문서 복사 및 PDF 저장이 실행됩니다.)
   * **액세스할 수 있는 사용자:** **`모든 사용자(Anyone)`**로 설정합니다. (경상남도교육청 계정이 없는 학부모 및 학생도 누구나 폼을 열고 서명할 수 있어야 하기 때문입니다.)
4. **배포** 버튼을 클릭합니다.

### ⑥ 구글 권한 승인 (최초 1회 필수)
1. 배포 중 **[액세스 승인]** 버튼 창이 뜨면 클릭합니다.
2. 본인의 구글 계정(도교육청 계정)을 선택합니다.
3. "Google에서 이 앱을 검증하지 않았습니다"라는 경고 창이 뜨면 당황하지 마시고, 왼쪽 아래의 **'고급'** 링크를 클릭한 후 **'제목 없는 프로젝트(으)로 이동(안전하지 않음)'**을 클릭합니다.
4. 개인 구글 드라이브와 문서 관리 권한을 허용하기 위해 **[허용]** 버튼을 클릭합니다.

### ⑦ 학부모 전송용 URL 주소 복사
1. 배포가 완료되면 화면에 **웹앱 URL** 주소가 나타납니다.
2. 이 URL 주소를 복사하여 스마트폰 문자 메시지, 알림톡(아이엠스쿨 등), 가정통신문 안내장에 **QR 코드** 형태로 제작하여 배포하시면 됩니다.

---

## 4. 관리 및 결과 확인

* 학부모가 링크를 통해 서명 후 제출하면, 구글 드라이브 지정 폴더에 **`[학생이름]_입학등록확인서.pdf`** 파일이 실시간으로 생성되어 쌓이게 됩니다.
* PDF를 다운로드하여 열어보시면, 템플릿 문서에 지정해 둔 위치에 학부모의 실제 손글씨 서명이 깔끔한 크기로 들어가 있는 것을 확인하실 수 있습니다.
